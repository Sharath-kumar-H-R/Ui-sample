Feature Enhancements Overview

1. Membership

This feature enables the definition of membership levels for customers based on their lifetime spending. It also provides configuration options for how loyalty points are earned and redeemed under each membership tier.
Click here to view documentation and UI example

2. Points Earning Logic

This logic is included within the Membership configuration:

  • Field Name: Loyalty Points Per Currency Unit (Rs)
  • Determines how many loyalty points are awarded per RM spent.
  • Also defines how points convert back to currency during redemption.
Examples:
• If 1 point = RM1, set this field to 1
• If 10 points = RM1, set this field to 10

3. Points Redemption

Members can redeem loyalty points via two methods:

  • a. Redeem for Vouchers: Convert points to vouchers (e.g., RM5 off on next purchase).
  • b. Redeem for Products: Exchange points for selected F&B items like drinks, snacks, etc.
Click here to view documentation and UI example

4. First-Time User Voucher – Buy 1 Free 1

A Buy 1 Free 1 voucher is automatically applied for first-time users upon completing their first purchase. This is system-driven and does not require manual handling.
Click here to view documentation and UI example

5. Complimentary Drink Voucher

A complimentary drink voucher is automatically issued when a customer:

  • Completes 8 cup purchases, or
  • Meets a configured condition such as:
    “Spends RM10 minimum for 5 consecutive days” → receives a free snack/product.

Reward rules are customizable via admin settings.
